
# Modélisation Statistique pour la Prédiction du Prix des Voitures

## Description du Projet

Ce projet vise à construire le meilleur modèle de régression possible pour prédire le prix d'un véhicule. L'analyse porte une attention particulière à l'identification et à la gestion des problèmes courants en régression, tels que la multicolinéarité, les valeurs aberrantes, et la normalité des résidus.

## Objectif Principal

Construire un modèle de régression optimisé pour prédire le prix des voitures en :
- Identifiant les variables explicatives les plus importantes
- Gérant les problèmes de multicolinéarité
- Détectant et traitant les valeurs aberrantes
- Validant les hypothèses de régression
- Optimisant les performances du modèle

## Technologies Utilisées

- **Python** : Langage de programmation principal
- **Pandas** : Manipulation et analyse des données
- **NumPy** : Calculs numériques
- **Scikit-learn** : Modèles de régression
- **Statsmodels** : Analyses statistiques avancées
- **Matplotlib/Seaborn** : Visualisations
- **Jupyter Notebook** : Environnement de développement

## Données

- **Variables** : Caractéristiques des véhicules (marque, modèle, année, kilométrage, etc.)
- **Variable cible** : Prix du véhicule
- **Type de problème** : Régression

## Méthodologie

### 1. Exploration des Données
- Analyse descriptive des variables
- Identification des corrélations
- Détection des valeurs manquantes

### 2. Préprocessing
- Traitement des valeurs manquantes
- Encodage des variables catégorielles
- Normalisation/standardisation si nécessaire

### 3. Analyse Statistique
- Tests de multicolinéarité (VIF)
- Détection des valeurs aberrantes
- Analyse des résidus
- Tests de normalité

### 4. Modélisation
- Régression linéaire multiple
- Sélection de variables (forward/backward selection)
- Validation croisée
- Optimisation des hyperparamètres

### 5. Évaluation
- Métriques de régression (R², RMSE, MAE)
- Validation des hypothèses
- Analyse des résidus
- Prédictions et intervalles de confiance

## Métriques d'Évaluation

- **R² (Coefficient de détermination)** : Qualité globale du modèle
- **RMSE (Root Mean Squared Error)** : Erreur quadratique moyenne
- **MAE (Mean Absolute Error)** : Erreur absolue moyenne
- **Validation croisée** : Robustesse du modèle
